What's new in REST API Version 16.0

IBM® QRadar® 7.4.3 introduces version 16.0 of the API endpoints.

New endpoints

RESTful API 16.0 introduces new categories of API endpoints and updates to existing endpoints in the following categories:

Configuration endpoints
Access authorized services
Certificates components
Event sources
Flow applications
Flow common destination ports
SIEM endpoints
Offenses
Staged Configuration endpoints
Certificates
Flow applications

Deprecated endpoints

All version 13.x API endpoints are marked as deprecated in QRadar 7.4.3.

Although deprecated endpoints continue to function, they will be removed in a future release. You must update your integration to use the most recent version of the QRadar RESTful API. Responses to deprecated endpoint requests include a Deprecated response header.
